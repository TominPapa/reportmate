# DM Automation Platform — 수익 모델

> 최종 업데이트: 2026-04-26

---

## 핵심 포지셔닝

**대상 고객:** Instagram DM 자동화를 Manychat 등 외부 도구로 운영하는 브랜드/마케터/에이전시  
**핵심 가치:** 발송 도구는 이미 있다. 없는 것은 **시나리오 설계 → 추적 → 분석 → 최적화** 워크플로다.  
**과금 기준:** 리드 수 + 워크스페이스 수 + 기능 접근 수준

## 2. 템플릿 마켓플레이스 (부수익)

### 개요
- 플랫폼에서 검증된 고품질 시나리오 템플릿을 유료 판매
- 판매자: 플랫폼 운영팀 or 인증된 파트너(에이전시)

### 과금 방식

| 유형 | 가격 | 설명 |
|------|------|------|
| 단건 구매 | $19 ~ $99 | 고급 시나리오 템플릿 1개 영구 소유 |
| 번들 팩 | $49 ~ $199 | 업종별 5~10개 템플릿 묶음 |
| 월정액 All-Access | $29/월 | 전체 프리미엄 템플릿 무제한 사용 |

### 수수료
- 파트너 템플릿 판매 시 수익의 30% 플랫폼 수수료
- 파트너에게 70% 지급

### 템플릿 등급 체계
- `basic`: 무료 제공
- `standard`: 구독 플랜 포함 or 단건 구매
- `advanced`: 단건 구매 또는 All-Access 전용

## 3. 에이전시 / 엔터프라이즈 플랜

### Agency 플랜
- **대상:** 복수 클라이언트 워크스페이스를 관리하는 마케팅 에이전시
- **가격:** $299/월 (연 결제 시 $249/월)
- **포함:**
  - 워크스페이스 25개
  - 클라이언트별 권한 분리
  - 화이트라벨 대시보드 (로고 교체)
  - 클라이언트 리포트 PDF 내보내기
  - 전담 온보딩 지원

### Enterprise 플랜
- **대상:** 대형 브랜드 인하우스 팀 (50,000+ 리드/월)
- **가격:** 맞춤 견적
- **포함:**
  - 워크스페이스 무제한
  - 리드 수 무제한
  - SSO (SAML/OIDC)
  - 전용 인프라 옵션
  - SLA 보장
  - 전담 Customer Success Manager

## 5. 수익화 로드맵

### Phase 1 (MVP 출시 ~ 3개월)
- **목표:** 제품-시장 적합성 검증
- **과금 방식:** Free + Starter만 운영
- **측정 지표:** 유료 전환율, 7일 유지율, NPS

### Phase 2 (3~6개월)
- **목표:** 수익 구조 안정화
- **추가 항목:** Growth / Pro 플랜 출시, 템플릿 마켓플레이스 베타
- **측정 지표:** MRR, 플랜별 이탈률, 초과 과금 비율

### Phase 3 (6~12개월)
- **목표:** 에이전시 채널 확장
- **추가 항목:** Agency 플랜, 파트너 프로그램, 리셀러 인증
- **측정 지표:** Agency ARR, 파트너 템플릿 판매 수

### Phase 4 (12개월 이후)
- **목표:** Enterprise 수주 및 글로벌 확장
- **추가 항목:** Enterprise 맞춤 계약, API 파트너십, 화이트라벨 OEM
- **측정 지표:** ACV(계약당 평균 가치), NRR(순 수익 유지율)

## 7. 핵심 과금 원칙

1. **리드 수 중심 과금** — 고객의 성공(리드 증가)이 우리 수익과 비례
2. **기능 잠금보다 볼륨 잠금 우선** — 핵심 기능은 모든 플랜에 오픈, 규모에서 차별화
3. **에이전시 채널 집중** — 에이전시 1곳이 여러 클라이언트를 온보딩 → 레버리지 높음
4. **마켓플레이스로 Lock-in 강화** — 고품질 템플릿이 플랫폼 이탈 비용을 높임
5. **Free 플랜은 영업 도구** — 과도하게 제한하면 입소문이 죽음
